# Justcogs

Discogs, but for Justin.

A mobile-first application for browsing and exploring your personal Discogs vinyl collection.

## Quick Start

1. **Set up your credentials:**
   - Copy `config.js.example` to `config.js`
   - Edit `config.js` and add your Discogs username and API token
   - Get your API token from https://www.discogs.com/settings/developers

2. **Start the server:**
   ```bash
   node server.js
   ```

3. **Open your browser:**
   Navigate to `http://localhost:3000`

## Features

- View your Discogs collection in a clean, grid layout
- See album artwork, artist, title, and year
- Collection statistics
- Genre filtering
- Community ratings (Discogs)
