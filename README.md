# Justcogs

A lightweight, opinionated personal catalogue inspired by Discogs.

Justcogs is a focused web project built to track and explore one's own music collection and listening interests. It is intentionally narrow in scope: fast, simple, and tailored rather than generic.

## Purpose

Discogs is powerful, but it is built for everyone. Justcogs is built for one person.

The goal of this project is to:

- Maintain a curated record of owned vinyl and related metadata
- Surface insights about the collection
- Experiment with lightweight data presentation patterns
- Serve as a playground for small product and engineering iterations
